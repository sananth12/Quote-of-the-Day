Quoter
======

Delivers the Quote of the Day to your terminal.

Backs up unread quotes, for you to read when offline.

##Install
```
$ npm install -g quoter
```

##Usage
To get the latest, today's quote:

```
$ quoter
``` 

When offline:

```
$ quoter --offline
```
